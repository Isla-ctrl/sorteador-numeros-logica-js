# Sorteador de Números

Projeto feito durante meus estudos de lógica de programação com JavaScript, pra treinar manipulação do DOM e controle de fluxo.

## Sobre o projeto

Um sorteador onde o usuário escolhe quantos números quer sortear e o intervalo (de/até). O app sorteia os números sem repetição e exibe o resultado na tela.

## O que eu aprendi

- Geração de números aleatórios dentro de um intervalo, com `Math.random()` e `Math.floor()`
- Uso de `while` pra evitar números repetidos no sorteio, verificando com `includes()` se o número já tinha saído
- Manipulação de classes CSS via JavaScript (`classList.add`/`remove`) pra habilitar e desabilitar o botão de reiniciar
- Organização do código em funções pequenas e específicas (sortear, obter número aleatório, alterar botão, reiniciar), cada uma com uma responsabilidade

## Tecnologias

- JavaScript (lógica e DOM)
- HTML/CSS (base do exercício)
